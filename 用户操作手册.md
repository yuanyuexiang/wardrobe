# 衣橱APP 用户操作手册

## 📱 应用简介

衣橱APP是一款基于React Native和Expo开发的服装商品管理应用，支持iOS、Android和Web平台。应用连接Directus后端系统，提供商品浏览、分类管理、店铺信息展示等功能。

## 🚀 快速入门

### 初次使用配置

当您首次打开应用时，需要进行初始配置：

#### 第一步：API连接配置
1. 打开应用后会自动进入配置页面
2. 输入以下信息：
   - **认证Token**: 您的Directus API访问令牌
   - **API基础URL**: Directus服务器地址（如：https://your-directus.com）
3. 点击"测试连接"验证配置是否正确
4. 连接成功后点击"下一步"

#### 第二步：选择店铺
1. 系统会自动获取您有权限访问的店铺列表
2. 从列表中选择您要管理的店铺
3. 点击"完成配置"进入主界面

## 📋 主要功能界面

### 1. 商品页面（主页）

**功能概述**：浏览和管理商品的核心页面

#### 界面布局：
- **顶部区域**：店铺信息展示
  - 店铺名称和评分
  - 轮播按钮：查看所有商品轮播
  - 设置按钮：重新配置应用
- **分类导航**：水平滚动的商品分类标签
  - "推荐商品"：显示最新商品
  - 其他分类：按商品类别筛选
- **商品列表**：水平滚动的商品卡片

#### 操作方法：
1. **浏览商品**：
   - 左右滑动查看更多商品
   - 每张卡片显示商品图片、名称、描述和价格
   
2. **分类筛选**：
   - 点击顶部分类标签切换不同类别商品
   - "推荐商品"显示按时间排序的最新商品
   
3. **查看详情**：
   - 点击任意商品卡片进入商品详情页
   
4. **轮播模式**：
   - 点击"轮播"按钮开启全屏商品展示
   - 自动切换商品图片，适合展示用途
   
5. **下拉刷新**：
   - 在商品列表区域下拉可刷新商品数据

### 2. 商品详情页面

**功能概述**：查看单个商品的详细信息

#### 界面元素：
- **图片展示区**：
  - 主图片展示（支持多图片横滑）
  - 圆点指示器显示图片数量
  - 点击图片可全屏预览
- **商品信息区**：
  - 商品名称和价格
  - 商品描述
  - 分类信息

#### 操作方法：
1. **浏览图片**：
   - 左右滑动查看多张商品图片
   - 点击图片进入全屏预览模式
   
2. **全屏预览**：
   - 支持手势缩放和滑动
   - 点击空白区域退出全屏
   
3. **返回**：
   - 使用顶部返回按钮回到商品列表

### 3. 商家页面

**功能概述**：展示店铺详细信息和相关内容

#### 界面内容：
- 店铺基本信息
- 店铺图片展示
- 联系方式和地址
- 经营状态

#### 操作方法：
1. **查看店铺信息**：
   - 滚动页面查看完整店铺资料
   - 点击图片可放大查看
   
2. **图片预览**：
   - 点击店铺图片进入全屏预览
   - 支持多图片浏览

### 4. 调试页面（开发用）

**功能概述**：用于测试网络连接和配置管理

#### 主要功能：
- **网络测试**：测试API连接状态
- **配置管理**：
  - 进入配置页面
  - 重置配置并重新设置
- **状态信息**：显示当前连接和配置状态

## ⚙️ 设置和配置

### 重新配置应用

如需更换店铺或修改连接设置：

1. **从主页进入配置**：
   - 点击主页右上角设置图标
   - 选择"重新配置"
   
2. **从调试页面进入**：
   - 切换到调试标签
   - 点击"进入配置页面"或"重置并重新配置"

3. **配置步骤**：
   - 重复初次配置的两个步骤
   - 可以修改API连接或选择不同店铺

### 应用设置说明

- **数据缓存**：应用会缓存商品图片以提升加载速度
- **自动刷新**：商品数据支持下拉刷新
- **离线浏览**：已加载的商品可离线查看
- **多设备同步**：配置保存在本地，需在每台设备上单独配置

## 🔧 常见问题解决

### 连接问题

**问题**：无法连接到服务器
**解决方法**：
1. 检查网络连接是否正常
2. 确认API地址格式正确（包含http://或https://）
3. 验证认证Token是否有效
4. 使用调试页面测试连接

**问题**：商品加载失败
**解决方法**：
1. 下拉刷新商品列表
2. 检查网络状态
3. 重新进入应用
4. 如持续出现，尝试重新配置

### 显示问题

**问题**：商品图片不显示
**解决方法**：
1. 检查网络连接
2. 等待图片加载完成
3. 重新进入页面

**问题**：商品列表为空
**解决方法**：
1. 确认选择的店铺有商品数据
2. 尝试切换不同分类
3. 检查用户权限设置

### 配置问题

**问题**：找不到店铺列表
**解决方法**：
1. 确认用户账户有创建店铺的权限
2. 检查认证Token权限范围
3. 联系管理员确认账户设置
